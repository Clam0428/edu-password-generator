# edu 密码字典生成器

在挖掘教育src时，由于学校的系统大多需要我们通过学号/工号进行登录，但一般的社工手段并不能得到完全可用的账号信息，多数时候我们只能获取部分信息，比如姓名、学号，因此对于登录信息的其他部分我们只能进行猜测，但基于常人设置密码的习惯，大多数密码简单、带有特征性，此工具通过结合社工获取的部分信息和常人密码设置的习惯进行密码生成。

## 功能特性

- **多维信息整合**：支持姓名（自动转拼音）、学号、手机号、生日、身份证号、邮箱
- **丰富密码模式**：姓名+生日、生日+姓名、纯生日、手机号、学号、身份证/邮箱、分隔符组合、后缀变体等 8 大类
- **智能日期爆破**：仅知年份时可遍历全年 365 天生成所有日期组合
- **身份证解析**：自动从 18 位身份证号提取生日和后 6 位
- **拼音变体**：全拼、首字母大写、全大写、仅首字母等
- **长度筛选**：可设置密码最小/最大长度
- **去重与反转**：自动去重，可选生成反转版本
- **GUI 界面**：基于 CustomTkinter 的现代化深色主题界面

## 适用场景

- 教育行业 SRC 渗透测试
- 校园网弱口令审计
- 授权安全评估

## 安装与运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python main.py
```

Windows 用户也可直接双击 `run.bat`。

## 依赖

- Python 3.9+
- [customtkinter](https://github.com/TomSchimansky/CustomTkinter) >= 5.2.0
- [pypinyin](https://github.com/mozillazg/python-pinyin) >= 0.51.0

## 组合模式说明

| 模式 | 说明 |
|------|------|
| 姓名拼音 + 生日 | 全拼/首字母 + 完整生日或四位年份（默认启用） |
| 生日 + 姓名拼音 | 完整生日/年份在前，姓名在后 |
| 纯生日 / 年份 | 完整生日、六位或四位年份 |
| 手机号相关 | 完整手机号或姓名+手机号 |
| 身份证 / 邮箱 | 身份证后六位、邮箱前缀 |
| 带分隔符组合 | 下划线、点号等少量常见分隔 |
| 学号相关 | 学号单独或与生日/姓名组合 |
| 常见后缀变体 | `!` `@` `#` `$` `123` 等后缀 |
